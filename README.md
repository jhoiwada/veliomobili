# veliomobili

# Panduan Menjalankan Website Velio Mobili

## 1. Menggunakan Laragon (Rekomendasi)

Karena folder proyek ini berada di direktori `www` Laragon (`C:\laragon\www\mobilivelio`):

1. Buka aplikasi **Laragon**.
2. Klik tombol **Start All**.
3. Buka browser dan kunjungi: **http://mobilivelio.test**

## 2. Membuka Langsung (Tanpa Server)

Website ini bersifat statis (HTML/CSS/JS), jadi Anda bisa menjalankannya tanpa server:

1. Buka File Explorer.
2. Arahkan ke folder proyek ini.
3. Klik dua kali pada file **`index.html`**.
4. Website akan terbuka di browser default Anda.

## 3. Menggunakan VS Code Live Server

Jika Anda menggunakan Visual Studio Code untuk mengedit:

1. Pastikan ekstensi **Live Server** sudah terinstall.
2. Klik kanan pada file `index.html` di sidebar.
3. Pilih **"Open with Live Server"**.

## Catatan

- Semua halaman (`tentang.html`, `layanan.html`, `galeri.html`, `kontak.html`) sudah saling terhubung.
- Pastikan koneksi internet aktif untuk memuat Google Fonts dan Font Awesome.
